# rd132211-bevy

A very simple remake of the oldest version available in the Minecraft launcher, [rd-132211](https://minecraft.wiki/w/Java_Edition_pre-Classic_rd-132211), using Bevy.

Try it out: https://skybird233.github.io/rd132211-bevy/

## Development

Build the project with:
```shell
cargo build
# or
cargo build --release
```

Tested on Linux x86_64 with X11 and the web platform.
